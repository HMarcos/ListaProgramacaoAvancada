ENUNCIADO:

O que fazem os seguintes programas em C?

int main(){
  int vet[] = {4,9,13};
  int i;
  for(i=0;i<3;i++){
    printf("%d ",*(vet+i));
  }
}

int main(){
  int vet[] = {4,9,13};
  int i;
  for(i=0;i<3;i++){
    printf("%X ",vet+i);
  }
}


////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

RESPOSTA:

O primeiro programa imprime o conteúdo associado a cada endereço relacionado à posição de cada elemento do vetor.

O segundo programa imprime o endereço associado a cada elemento do vetor.
